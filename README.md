## 团购H5 5.8开发文档

### 安装

+ 克隆源码到本地
	>git clone git@git.dev.sh.ctripcorp.com:xuweichen/tuan-app-5-8.git webapp
	`文件夹名字必须为webapp，本地打hybrid包grunt只认这个目录` 
+ 进入_runtime文件夹
+ 执行npm install
+ 启动服务node start-server.js
+ 访问http://localhost:3000/webapp/tuan/index.html#home
